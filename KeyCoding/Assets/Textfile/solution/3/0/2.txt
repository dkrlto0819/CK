a=[3, 6, 7, 81, 9]
// 데이터가 5개 있는 배열
for i in a:
		print(a[i])

/*
for문이란? 정해진 횟수나 특정한 번호부터 번호까지 반복시키고 싶을 때 사용합니다.
기본 형태는 아래와 같습니다.
-------------------------------------------------------------
    for 변수명 in range(초기값, 마지막 값):
-------------------------------------------------------------

배열이란? 같은 타입의 자료가 여러 개 열거된 자료를 말합니다.
기본 형태는 아래와 같습니다.
-------------------------------------------------------------
    배열이름 = [저장할 자료들];
-------------------------------------------------------------
*/
