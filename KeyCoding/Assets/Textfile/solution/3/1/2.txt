switch=0
for i in range(2, 51):
		for j in range(2, i/2+1):
			if i%2==0 :
				switch=1
				break
		if switch==0 :
			print(i)

/*
for문이란? 정해진 횟수나 특정한 번호부터 번호까지 반복시키고 싶을 때 사용합니다.
기본 형태는 아래와 같습니다.
-------------------------------------------------------------
    for 변수명 in range(초기값, 마지막값):
-------------------------------------------------------------

이중 for문이란? for문 안에 또다른 for문이 있는 형태로 안쪽에 있는 for문이
모두 돌아야만 바깥의 for문이 한번 실행이 완료되는 형태를 가지고 있습니다.

i%j ? i를 j로 나눈 나머지를 이야기 합니다.

if문이란? 조건에 따라 실행할 문장을 나누는 함수입니다.
기본 형태는 아래와 같습니다.
-------------------------------------------------------------
    if 조건 :
    elif 조건 :
    else :
-------------------------------------------------------------
*/
